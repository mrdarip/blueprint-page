What technical features does the style contais?

The page is responsive, try resizing! Appart from that, the style features mostly general rules, allowing you to write your html without worrying about the style, the only thing you need to do is to add the class "moving-check-bg" to the body tag, and the background will start moving! If you wanted too you could add it to a section and it would feature it as well!

That's convenient! You can change the colors of the background by changing the values of the variables --white and --blue, and you can change the speed of the animation by changing the value of the variable animation-duration, the value is in seconds, so if you want to make it slower, you can change it to 5s, or if you want to make it faster, you can change it to 1s!
